Project Overview

The Unique Expense Manager (UEM) app tracks and manages daily, weekly, and monthly expenses. It is built using Java for the frontend, with Google Firebase for backend data storage. The app allows users to log their income, categorize expenses, and receive notifications when they approach their spending limit.

Features

  • User Login: Secure login using Google account
  • Expense Categories: Users can categorize expenses (e.g., Food, Rent, Movies)
  • Expense Analysis: Visual analysis of daily, weekly, and monthly expenses
  • Data Privacy: User data securely stored in Google Firebase
  • Notifications: Alerts when spending limits are reached

Tools Used

Android Studio (Java), Google Firebase
