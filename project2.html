Project Overview

This system automates the generation of question papers using Artificial Intelligence techniques like NLP and machine learning. It leverages Python for backend processing and HTML, CSS, and JavaScript for frontend display. The system allows educators to create dynamic, customized exam papers, saving time and reducing bias.

Features

  • AI-driven question generation with customizable difficulty and topic selection
  • NLP techniques ensure grammatically correct and diverse questions
  • Automated generation of subjective and objective questions
  • Seamless integration with web technologies for a user-friendly interface

Tools Used

Python, HTML, CSS, JavaScript, Flask, MySQL, NLP Libraries
